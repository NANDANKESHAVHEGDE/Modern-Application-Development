This is a brief read on things that can help you to run the code successfully 

a.Navigate to the root folder - GroceryStore_app and create a virtual environment and install the pre-requisite libraries as follows
i. python -m venv <virtual_environment_name>
ii.pip install -r Requirements.txt

Windows Users to activate virtual environment-
b.From the root execute - .\<virtual_environment_name>\Scripts\activate.bat

Ubuntu or any Linux based Desktop version users to activate the virtual environment -
b.From the root execute source ../<virtual_environment_name>/bin/activate
	
c.Execute python main.py from inside the GroceryStore root folder to start the application 
	
These steps are highly recommended to run the code in a seamless manner
